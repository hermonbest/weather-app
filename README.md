# Weather App

This is a simple and beautiful weather application that provides the current weather and a 5-day forecast for any city.

## Features

- **City Search:** Find the current weather and forecast for any city in the world.
- **Current Weather:** Displays the temperature, feels like temperature, humidity, wind speed, and visibility.
- **5-Day Forecast:** Shows the weather forecast for the next 5 days, including the date, weather icon, high/low temperatures, and a short description.
- **Responsive Design:** The application is fully responsive and works on all devices.
- **Glassmorphism UI:** A modern and sleek user interface with a "glassmorphism" effect.

## Technologies Used

- **HTML:** The structure of the web page.
- **CSS:** Styling the application with a modern design, including gradients and a glassmorphism effect.
- **JavaScript:** Fetching data from the Weather API and dynamically updating the UI.
- **Webpack:** Bundling the JavaScript modules.
- **Visual Crossing Weather API:** Used to fetch the weather data.

## Setup and Run

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/weather-app.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd weather-app
   ```

3. **Install the dependencies:**
   ```bash
   npm install
   ```

4. **Run the application:**
   ```bash
   npm start
   ```

The application will be available at `http://localhost:8080`.
